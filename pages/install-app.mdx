This guide will walk you through the process of installing the Gitsecure GitHub App to enhance the security of your code repositories on GitHub.


## Step 1: Navigate to the Installation Page

1. Click the "Install App" button below to visit the Gitsecure GitHub App installation page.

## Step 2: Authorize Access

Once you are on the GitHub App installation page, follow these instructions:

- Ensure that you are logged into your GitHub account.
- Select the organization where you want to enable Gitsecure.

![Gitsecure](/screenzy-1696314977442.png)


## Step 3: Configuration (if necessary)

Depending on your requirements, you may need to configure Gitsecure for specific repositories. For detailed configuration instructions, please refer to the Gitsecure documentation.

![Gitsecure](/screenzy-1696314991980.png)


## Step 4: Confirm Installation

Before proceeding, review the permissions and access levels requested by Gitsecure. Ensure you are comfortable with the permissions.


## Step 5: Complete the Installation

To finalize the installation process, follow these steps:

1. Click the "Install" button.
2. You will be redirected to your dashboard on Gitsecure.
