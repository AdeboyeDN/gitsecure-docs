To enhance the security of your code repositories on GitHub, you can install Gitsecure Github App. Follow these steps to get started:


## 1. Navigate to the Installation Page:

Click the "Install App" button below to visit the Gitsecure GitHub App installation page.

![Gitsecure](/screenzy-1696314977442.png)

## 2. Authorize Access:

You will be directed to the GitHub App installation page. If prompted, ensure you are logged into your GitHub account and select the repository where you want to enable Gitsecure.
![Gitsecure](/screenzy-1696314991980.png)

## 3. Configuration (if necessary):

Depending on your needs, you may need to configure Gitsecure for specific repositories. Refer to the Gitsecure documentation for detailed configuration instructions.

## 4. Confirm Installation:

Review the permissions and access levels requested by Gitsecure. Make sure you are comfortable with the permissions before proceeding.

## 5. Complete the Installation:

Click the "Install" button to finalize the installation process.
![Gitsecure](/screenzy-1696315005385.png)
