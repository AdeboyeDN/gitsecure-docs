Learn how to seamlessly install Wasps in Visual Studio Code by following these simple steps.

### 1. Open the Extensions View in VS Code:
- Click on the Extensions icon in the Activity Bar on the side of the window.
- Alternatively, you can open the Extensions view by pressing `Ctrl+Shift+X`.

### 2. Search for "Wasps":
- In the Extensions view search bar, type "Wasps".
- Look for the extension named "Wasps" in the search results.

### 3. Install the Wasps Extension:
- Click the "Install" button on the Wasps extension from the search results.
- Wait for the installation to complete. You will need to reload VS Code to activate the extension.
