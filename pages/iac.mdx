# Infrastructure as code (IaC) Security

Gitsecure scans cloud infrastructure configurations to find misconfigurations before they're deployed.

## Supported clouds and frameworks

The table below provides a comprehensive list of clouds and frameworks compatible with Gitsecure's IaC engine.

| Frameworks         | Supported |
| ------------------ | --------- |
| Terraform          | Yes       |
| Terraform plan     | Yes       |
| CloudFormation     | Yes       |
| Kubernetes         | Yes       |
| ARM Templates      | Yes       |
| Serverless         | Yes       |
| Helm               | Yes       |
| AWS CDK            | Yes       |
